Accessibility

I'm committed to making this website accessible to everyone.

Accessibility Features

This website has been built with accessibility in mind:

  • Semantic HTML structure
  • Sufficient color contrast ratios
  • Responsive design for all screen sizes
  • No JavaScript required for core functionality
  • Keyboard navigable
  • Screen reader friendly

Standards

This website aims to conform to the Web Content Accessibility Guidelines (WCAG) 2.1 Level AA standards.

Feedback

If you encounter any accessibility barriers on this website, please let me know. I take accessibility seriously and will work to address any issues.

Continuous Improvement

Accessibility is an ongoing effort. I regularly review and improve this website to ensure it remains accessible to all users.
